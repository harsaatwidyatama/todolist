#Aplikasi To-Do List

## Deskripsi
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian

## Fitur
- Tambah tugas
- Tandai tugas
- Edit tugas
- Hapus tugas

## Struktur Folder

- 'index.php' - halaman utama

## Cara Menjalankan

1. Salin folder ke 'htdocs/'
2. Jalankan XAMPP dan buka lalu ketik di browser 'localhost/todolist'
